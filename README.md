🩺 FilaSaúde – Sistema Simples de Prioridade de Atendimento

Projeto desenvolvido em linguagem C com foco em lógica de programação, estruturas condicionais e aplicação social.

📌 Sobre o Projeto

O FilaSaúde foi criado com o objetivo de auxiliar na organização inicial de atendimentos em ambientes comunitários e unidades básicas de saúde, utilizando critérios simples para indicar a prioridade de atendimento de um paciente.

O programa realiza uma análise básica considerando:

idade do paciente;
temperatura corporal;
presença de falta de ar.

A partir dessas informações, o sistema classifica a prioridade do atendimento em:

🔴 Alta
🟡 Moderada
🟢 Baixa
🎯 Objetivo

Aplicar os conceitos fundamentais de programação em C utilizando:

estrutura sequencial;
comandos condicionais (if, else if, else);
entrada e saída de dados.

O projeto foi desenvolvido sem o uso de:

laços de repetição;
funções;
estruturas avançadas.
🧠 Tecnologias Utilizadas
Linguagem C
Visual Studio Code
GCC (GNU Compiler Collection)
⚙️ Funcionalidades

✔ Cadastro simples de informações do paciente
✔ Análise de prioridade de atendimento
✔ Exibição automática do resultado
✔ Estrutura simples e didática

📋 Regras de Prioridade
Condição	Resultado
Temperatura ≥ 39°C ou falta de ar	Prioridade ALTA
Idade ≥ 60 anos ou febre moderada	Prioridade MODERADA
Demais casos	Prioridade BAIXA
▶️ Como Executar
Compilar o programa
gcc projeto.c -o projeto01
Executar o programa
.\projeto01
💡 Exemplo de Execução
=== Sistema Simples de Prioridade de Atendimento ===

Informe a idade do paciente: 12
Informe a temperatura corporal: 39
Possui falta de ar? (1 = Sim / 0 = Nao): 0

=== RESULTADO ===
Prioridade ALTA de atendimento.
🌍 Impacto Social

O projeto busca demonstrar como soluções computacionais simples podem auxiliar a comunidade na organização de atendimentos e na conscientização sobre prioridades em situações básicas de saúde.

Além disso, o sistema possui caráter educativo, podendo ser utilizado em:

escolas;
oficinas de programação;
projetos comunitários;
atividades introdutórias de lógica computacional.
